## 1. Routine du Matin 🌞

### Le bon ordre chaque matin

**🔐 Connexion → 📊 Dashboard → ✅ Actions → 📞 Salons → 📧 Outreach → 🔄 Pipeline**

### Checklist express

- [ ] Se connecter
- [ ] Consulter le Dashboard
- [ ] Vérifier les Actions
- [ ] Consulter les Salons prioritaires
- [ ] Vérifier l’Outreach
- [ ] Mettre à jour le Pipeline

### 1) Se connecter 🔐

- Ouvrir **laloge-crm-v2.vercel.app**
- Entrer votre **email** et votre **mot de passe**
- Cliquer sur **Se connecter**
- Si vous êtes sur téléphone, ouvrir le **menu** pour accéder aux pages

### 2) Consulter le Dashboard 📊

Quand vous arrivez sur le CRM, commencez toujours par le **Dashboard**.

À regarder en priorité :
- le nombre de salons du jour
- les **actions en attente**
- le **brief du matin**
- les salons prioritaires
- les indicateurs importants du pipeline

👉 Si quelque chose est marqué **urgent**, commencez par là.

### 3) Vérifier les Actions ✅

Aller sur la page **Actions**.

C’est ici que vous voyez les propositions faites par les agents IA.

Ce que vous devez faire :
- lire la proposition
- cliquer sur **Approuver** si elle vous semble correcte
- cliquer sur **Rejeter** si elle n’est pas adaptée

👉 En général, il vaut mieux traiter les actions en attente tôt dans la journée.

### 4) Consulter les Salons prioritaires 📞

Aller sur la page **Salons**.

Ensuite :
- utiliser les filtres
- regarder les salons avec **score > 40**
- repérer les meilleurs profils à appeler en premier

À vérifier avant un appel :
- le nom du salon
- la ville
- le téléphone
- le score
- le statut actuel
- l’historique de la fiche

### 5) Vérifier l’Outreach 📧

Aller sur la page **Outreach**.

Cette page permet de voir :
- les emails planifiés
- les campagnes en cours
- les réponses reçues
- l’historique des communications

👉 Si un salon a répondu, pensez à reprendre la fiche salon pour continuer le suivi.

### 6) Mettre à jour le Pipeline 🔄

Après chaque appel ou retour important :
- ouvrir la fiche du salon
- mettre à jour son **statut commercial**
- ajouter une note si nécessaire

Les statuts les plus utilisés :
- **Nouveau**
- **Contacté**
- **Intéressé**
- **RDV planifié**
- **Négociation**
- **Gagné**
- **Perdu**
- **Client actif**

👉 Le plus important : **ne pas attendre la fin de journée** pour mettre à jour les statuts.

## 3. Glossaire 📚

### **Score**
Note de qualité du salon, de **0 à 100**.
Elle est calculée automatiquement à partir de plusieurs éléments : avis Google, présence en ligne, taille d’équipe, etc.

### **Enrichissement**
Quand un agent IA complète automatiquement les informations d’un salon.

### **Action en attente**
Une proposition d’un agent IA qui a besoin d’une validation humaine.

### **Pipeline**
L’entonnoir commercial : de **Nouveau** à **Client actif**.

### **Outreach**
Les communications envoyées aux salons : emails, messages, relances.

### **Dossier**
Le rapport de présentation créé pour proposer un salon à une marque.

## 4. FAQ ❓

### Je ne retrouve plus un salon
➡️ Aller dans **Salons** et utiliser la **recherche** ou les **filtres**.

### Comment approuver une action IA ?
➡️ Aller dans **Actions**, ouvrir la proposition, puis cliquer sur **Approuver**.

### Comment changer le statut d’un salon ?
➡️ Ouvrir la **fiche salon**, puis mettre à jour la section **Pipeline**.

### Qui contacter si j’ai un problème ?
➡️ **Ludovic** ou le **support La Loge**.
