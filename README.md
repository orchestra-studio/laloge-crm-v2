# La Loge CRM v2

CRM vertical beauté pour La Loge — gestion de salons de coiffure et relations marques.

## Stack

- Next.js 16 + React 19
- Tailwind CSS v4 + ShadCN UI
- Supabase (auth, database, realtime)
- TanStack Table

## Développement

```bash
npm install
npm run dev
```

## Build

```bash
node scripts/build.mjs
```

## Déploiement

Push vers `main` → auto-deploy Vercel.

- **Production** : https://laloge-crm-v2.vercel.app
